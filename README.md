# XamlBrewer WinUI3 LiveCharts2 Sample

This WinUI 3 Desktop app demonstrates [LiveCharts2](https://livecharts.dev/), a Skia-based cross-platform package for charts, maps, and gauges in .NET.

Features:
* Hello World,
* Sample gallery,
* Theming, and
* How to plot functions
